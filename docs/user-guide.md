# Panduan Pengguna

Panduan ini ditujukan untuk pengguna aplikasi Sistem Manajemen Dokumen Akuntansi.

## Role Pengguna

### 1. Admin Akuntansi

Hak akses:
- Melihat dashboard
- Melihat Document Management
- Upload dokumen baru
- Edit dokumen
- Hapus dokumen (ke riwayat)
- Restore dokumen dari riwayat
- Hapus permanen dokumen dari riwayat

### 2. Staff Akuntansi

Hak akses:
- Melihat dashboard
- Melihat Document Management
- Melihat dokumen
- Unduh dokumen

Catatan:
- Staff tidak memiliki akses ke menu Upload dan File History.

## Alur Penggunaan Utama

### Login
1. Buka halaman login.
2. Masukkan username dan password.
3. Klik tombol login.

### Upload Dokumen (Admin)
1. Buka menu `Upload Document`.
2. Isi data dokumen:
   - Nama SPPD
   - Tanggal
   - Kategori
3. Pilih file dokumen.
4. Klik `Upload`.

### Kelola Dokumen
1. Buka menu `Document Management`.
2. Gunakan fitur:
   - Pencarian nama/kategori
   - Filter tanggal
   - Filter kategori
3. Aksi dokumen:
   - `Lihat Dokumen` untuk membuka file di tab baru
   - `Edit Dokumen` (Admin)
   - `Hapus Dokumen` (Admin, dipindah ke riwayat)

### Kelola File History (Admin)
1. Buka menu `File History`.
2. Pilih satu atau lebih dokumen terhapus.
3. Gunakan aksi:
   - `Restorasi File Ini` untuk mengembalikan dokumen
   - `Hapus Permanen File Ini` untuk menghapus permanen

## Praktik Penggunaan yang Disarankan

- Gunakan nama dokumen yang konsisten dan jelas.
- Validasi kategori sebelum upload.
- Gunakan fitur pencarian/filter sebelum melakukan restore atau delete permanen.
